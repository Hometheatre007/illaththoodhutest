- Prompt writing
- Debugging with AI
- Code optimization
- API integration
- Automation building
- Documentation writing
- Error analysis
- UI/UX suggestions

---

## Use AI with Small Token Prompts

### Good Prompt Format

```
Role + Task + Input + Output Format + Limits
```

### Example

```
Act as Python dev. Fix this code. Return only code. Keep short.
```

---

## Token Saving Tips

- Be direct
- Use short context
- Ask one task at a time
- Request bullet output
- Ask only changed code
- Reuse previous context
- Avoid long story prompts

## Best Workflow

1. Plan project modules
2. Ask AI for one module
3. Test manually
4. Fix errors with AI
5. Optimize final code

---

## Prompt Templates

### Build Feature
```
Create login page in React. Clean UI. Single file.
```

### Fix Bug
```
Find bug in this JS code. Explain shortly. Give fixed code only.
```

### Optimize
```
Reduce memory and speed up this Python script.
```

---

## Security Note

- Do not give AI direct terminal access unless fully trusted.
- Review every command before running.
- Use sandbox/VM for risky tasks.
- Never share passwords, API keys, or private data.
- Keep least-privilege access.

---

## Final Rule

> Use AI as assistant, not replacement. Verify output before production.
